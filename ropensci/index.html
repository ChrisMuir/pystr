What is pystr?

A port of (most) Python string methods to R

str.method() –> pystr_method()

For example…

str.count(substr) –> pystr_count(str, substr)

It's on CRAN

install.packages("pystr")

Why did I build pystr?

Split a string

The Python Way

url = "www.example.com"
print(url.split("."))
## ['www', 'example', 'com']

Split a string

The R Way

url = "www.example.com"

strsplit(url, ".")
## [[1]]
##  [1] "" "" "" "" "" "" "" "" "" "" "" "" "" "" ""

Ugh. ?strsplit

url = "www.example.com"

strsplit(url, ".", fixed=TRUE)
## [[1]]
## [1] "www"     "example" "com"

Split a string

The pystr Way

url = "www.example.com"

pystr_split(url, ".")
## [[1]]
## [1] "www"     "example" "com"

Check if a string begins with a substring

The Python Way

url = "http://www.neo4j.com"
print(url.startswith("https://"))
## False

Check if a string begins with a substring

The R Way

url = "http://www.neo4j.com"
substring(url, 1, nchar("https://") - 1) == "https://"
## [1] FALSE

Check if a string begins with a substring

The pystr Way

url = "http://www.neo4j.com"
pystr_startswith(url, "https://")
## [1] FALSE

Substitute the first n occurrences of a substring

The Python Way

s = "All I do is win, win, win no matter what..."
print(s.replace("win", "lose", 2))
## All I do is lose, lose, win no matter what...

Substitute the first n occurrences of a substring

The R Way

s = "All I do is win, win, win no matter what..."
count = 0

while (count < 2) {
  s = sub("win", "lose", s)
  count = count + 1
}

s
## [1] "All I do is lose, lose, win no matter what..."

Substitute the first n occurrences of a substring

The pystr Way

s = "All I do is win, win, win no matter what..."
pystr_replace(s, "win", "lose", 2)
## [1] "All I do is lose, lose, win no matter what..."

Determine which substring occurs latest in the string

The Python Way

s = "3 + 3 + 1 > 5"
print(s.rfind(">") > s.rfind("+"))
## True

Determine which substring occurs latest in the string

The R Way

s = "3 + 3 + 1 > 5"
max(gregexpr(">", s)[[1]]) > max(gregexpr("\\+", s)[[1]])
## [1] TRUE

Determine which substring occurs latest in the string

The pystr Way

s = "3 + 3 + 1 > 5"
pystr_rfind(s, ">") > pystr_rfind(s, "+")
## [1] TRUE

Format a string

The Python Way

s = "The name is {last}. {first} {last}."
print(s.format(first="James", last="Bond"))
## The name is Bond. James Bond.

Format a string

The R Way

first = "James"; last = "Bond"
paste0("The names is ", last, ". ", first, " ", last, ".")
## [1] "The names is Bond. James Bond."
first = "James"; last = "Bond"
sprintf("The name is %s. %s %s.", last, first, last)
## [1] "The name is Bond. James Bond."

Format a string

The pystr Way

s = "The name is {last}. {first} {last}."
pystr_format(s, first="James", last="Bond")
## [1] "The name is Bond. James Bond."

What is it?

Rcpp integrates C++ into R easily.

Example

// [[Rcpp::export]]
IntegerVector pystr_count_(CharacterVector strs, std::string sub, int start, int end) {
  int input_size = strs.size();
  IntegerVector output(input_size);

  for(int i = 0; i < input_size; i++){
    if(strs[i] == NA_STRING){
      output[i] = NA_INTEGER;
    } else {
      output[i] = str_count(Rcpp::as<std::string>(strs[i]), sub, start, end);
    }
  }

  return output;
}

Example

pystr_count <- function(str, sub, start=1, end=max(nchar(str))) {
  return(pystr_count_(str, sub, start - 1, end - 1))
}
